ABOUT COURSEWORK

This course work is intended to teach the technologies used in web development. Primarily we are gonna work HTML5,CSS3, and JavaScript. This coursework is intended to focus on front-end technologies.

Below are the topics which are going to covered in this coursework.

  • AJAX technologies for asynchronously loading content.
